---
title: getMPIRankCount
---

Returns the number of physical MPI ranks in the simulation 

## Syntax
```python
sst.getMPIRankCount()
```

## Parameters
* **returns** number of MPI ranks in the simulation 

## Example

```python
import sst

ranks = sst.getMPIRankCount()
```

## Import
```python
import sst
```
