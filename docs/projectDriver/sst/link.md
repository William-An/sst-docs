### Remarks

## Requirements

```python
 import sst
```

## Syntax

```python
link = sst.Link("linkName")
```

## Parameters

**linkName** - A name for the link.

## Return Value

**link** - a link object. You can use it to link two components together.

## Examples

### Example 1
```python
link = sst.Link("MyLink")
```
